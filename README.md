# 拾光循迹

校园失物招领小程序。基于微信小程序云开发，支持失物招领的登记、审核、检索与状态追踪。

## 功能

- 失物/招领登记：填写物品信息、上传图片、留下联系方式
- 审核管理：管理员审核内容，控制展示状态
- 线索检索：按类型（招领/寻物）、分类、状态、关键词筛选
- 状态追踪：从提交、审核、展示到完成的完整流转
- 我的登记：查看个人提交的所有记录及审核状态
- 暗色模式：跟随系统主题切换

## 技术栈

- 微信小程序原生开发
- 微信小程序云开发（云函数、云数据库）
- CSS 自定义属性主题系统（明暗双主题）
- 毛玻璃 UI + 渐变背景

## 项目结构

```
miniprogram/
├── pages/          # 页面（首页/列表/发布/详情/我的/管理/我的登记）
├── custom-tab-bar/ # 自定义底部导航栏
├── styles/         # 全局样式（主题变量）
├── images/         # SVG 图标
├── utils/          # 工具函数（云调用、数据配置、本地存储、主题管理）
├── app.js          # 应用入口
├── app.json        # 应用配置
└── app.wxss        # 全局样式

cloudfunctions/     # 云函数
```

## 开始使用

1. 在微信开发者工具中导入项目
2. 在 `app.js` 中配置你的云环境 ID
3. 在微信开发者工具中关联云开发数据库
4. 编译运行
