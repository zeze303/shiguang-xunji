# 开发者工具检查清单

每次修改完成后，请按以下顺序在微信开发者工具中逐项检查。

---

## 1. 编译运行

- [ ] 开发者工具编译无报错（Console 无红色 error）
- [ ] 页面正常渲染，无白屏、无布局错位
- [ ] 所有图片/SVG 正常显示，无 404 加载失败

---

## 2. 页面逐个检查

### 首页（home）
- [ ] hero 头部显示完整，无布局挤压
- [ ] 三个按钮（登记招领 / 登记寻物 / 浏览台账）图标正常显示
- [ ] 统计四卡片 SVG 图标正常
- [ ] 快捷筛选 chips 有图标，可点击跳转正常
- [ ] 快捷筛选卡片（大卡片）图标 + 标题同行排列
- [ ] 最新信息列表正常展示

### 列表页（list）
- [ ] 类型 tab 切换正常（全部 / 招领 / 寻物）
- [ ] 分类 chip 轮廓样式正确，SVG 图标显示
- [ ] 状态 tag 文字标签样式正确
- [ ] 卡片内左侧 44×44 分类 SVG 图标显示
- [ ] 卡片底部"查看详情"轮廓按钮可点击跳转
- [ ] 搜索输入正常

### 详情页（detail）
- [ ] status-panel 三标签（状态/地点/时间）SVG 图标显示
- [ ] info-panel 四行（物品/地点/登记人/时间）图标显示
- [ ] contact-panel 电话图标 + 复制按钮正常
- [ ] 操作栏在内容短时不遮挡底部，内容超出时吸底
- [ ] 状态流转图标正确（待审/完成/审核通过）
- [ ] 图片可预览放大

### 管理页（admin）
- [ ] 卡片左侧色带颜色正确（待审橙 / 通过绿 / 驳回红）
- [ ] audit-badge 颜色正确
- [ ] 操作按钮颜色区分明显（通过绿填 / 驳回橙廓 / 删除红廓）

### 发布页（publish）
- [ ] 输入框聚焦时左侧出现 3rpx 青色指示条 + 边框变色
- [ ] 分类 picker 选中后显示对应 SVG 图标
- [ ] 表单布局完整，无错位
- [ ] 提交按钮 loading 状态正常

### 我的（mine）
- [ ] 头像 112rpx 正常显示
- [ ] 更新头像昵称 按钮可正常弹窗
- [ ] 我的登记菜单项左侧有渐变色带
- [ ] 审核管理中心卡片有橙色左边框
- [ ] 菜单项左侧有 SVG 图标

### 我的登记（my-posts）
- [ ] 卡片顶部状态色条正确（橙=审核中 / 蓝=展示中 / 绿=完成 / 红=拒绝）
- [ ] Tab 选中态有下划线 + 加粗，未选中为灰色文字
- [ ] 搜索输入正常

---

## 3. 交互与状态

- [ ] 页面滚动流畅，无卡顿
- [ ] 按钮点击有反馈（如跳转、弹窗正常）
- [ ] 空状态页面显示正常
- [ ] 数据加载中/失败时 UI 不崩溃
- [ ] TabBar 切换时选中项有底部青色圆点浮现（0.3s 过渡）

---

## 4. 暗色模式

- [ ] 切换到暗色模式，所有修改页面正常显示
- [ ] 语义色（绿/橙/红）在暗色模式下可辨识
- [ ] SVG 图标在暗色模式下清晰可见

---

## 5. 边界情况

- [ ] 超长标题/描述不溢出卡片边界
- [ ] 无图片时卡片布局不变形
- [ ] 数据为空时显示空状态占位

---

## 反馈格式

测试发现问题后直接告诉我，例如：

> 首页 hero 按钮图标偏左，超出容器边界
> 列表页暗色模式下分类图标看不清
> 管理页通过按钮文字颜色与背景对比度不够

我会修复后重新 commit 推送。
